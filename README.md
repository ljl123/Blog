# Blog
# 简介
基于 Spring Boot + Vue 的前后端分离博客系统 【根据教程练手的项目】 https://juejin.cn/post/6844903823966732302
# 后端
* 核心框架：Spring Boot
* 安全框架：shiro
* Token 认证：jwt
* 持久层框架：MyBatisPlus
* java版本：JDK8
# 前端
* 核心框架：Vue、Element UI、axios
* 编辑：markdown
# 功能
* 主页（按时间排序查看所有博客）
* 分类（分类查看博客）
* 发表博客、修改博客
* 关于我
* 登录（没做注册）
